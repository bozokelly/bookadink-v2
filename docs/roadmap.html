🔴 Blocker — Supabase RLS Profile Update Policy

The Users can update own profile RLS policy is missing WITH CHECK ((auth.uid() = id)). Without it, PostgREST upserts to the profiles table silently fail.

-- Run in Supabase SQL editor: DROP POLICY IF EXISTS "Users can update own profile" ON profiles; CREATE POLICY "Users can update own profile" ON profiles FOR UPDATE TO authenticated USING (auth.uid() = id) WITH CHECK (auth.uid() = id);
⚠ Verify by editing your profile in the app after applying — check the Supabase table editor to confirm the row changed.
